# Pipeline de Limpeza de Dados em Excel

## Objetivo

Realizar a limpeza e o tratamento de dados provenientes de uma planilha Excel com dados de uma empresa fictícia, corrigindo inconsistências, tratando valores ausentes e padronizando informações para garantir dados confiáveis para análise.

## Tecnologias Utilizadas

- Microsoft Excel
- Python
- Jupyter Notebook
